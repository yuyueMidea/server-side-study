## Node.js 常见服务框架总览
| 框架             | 语言特性  | 风格            | 适合人群 / 场景         |
| -------------- | ----- | ------------- | ----------------- |
| **Express.js** | JS    | 轻量、灵活         | 初学者、小型项目、REST API |
| **Koa.js**     | JS    | 极简、现代         | 中高级开发者，需要更自由控制    |
| **NestJS**     | TS    | 企业级、模块化       | 大型项目、团队协作、微服务     |
| **Fastify**    | JS/TS | 高性能、插件化       | 追求性能的中大型项目        |
| **Hapi.js**    | JS    | 配置驱动、安全强      | 企业后台、需要严格约束       |
| **AdonisJS**   | TS    | Laravel风格全栈框架 | 一体化开发（ORM+验证+中间件） |
| **MeteorJS**   | JS    | 前后端一体化        | 实时应用、全栈JS项目       |
| **Sails.js**   | JS    | MVC结构、传统Web   | 有传统MVC思维的开发者      |

二、总结对比表
| 框架           | 难度   | 性能   | 类型   | 特点               | 推荐场景   |
| ------------ | ---- | ---- | ---- | ---------------- | ------ |
| **Express**  | ⭐    | ⭐⭐   | 极简   | 教学入门、生态强         | 小型服务   |
| **Koa**      | ⭐⭐   | ⭐⭐⭐  | 极简现代 | async/await 极简自由 | 中型服务   |
| **NestJS**   | ⭐⭐⭐⭐ | ⭐⭐⭐  | 企业级  | 模块化、TS 强类型       | 大型系统   |
| **Fastify**  | ⭐⭐   | ⭐⭐⭐⭐ | 高性能  | 高速、插件化           | 高并发API |
| **Hapi**     | ⭐⭐⭐  | ⭐⭐   | 企业安全 | 配置驱动             | 企业后台   |
| **AdonisJS** | ⭐⭐⭐  | ⭐⭐   | 全家桶  | ORM+Auth一体化      | 中大型Web |
| **SailsJS**  | ⭐⭐   | ⭐    | MVC  | 传统架构             | 后台快速开发 |

三、选择建议
| 目标                | 推荐框架         | 理由             |
| ----------------- | ------------ | -------------- |
| 新人入门 / 教学         | **Express**  | 生态广、资料多        |
| 想追求更优雅代码风格        | **Koa**      | async/await、轻量 |
| 企业级项目 / 大团队       | **NestJS**   | 结构清晰、TS支持      |
| 高性能微服务            | **Fastify**  | 性能优越、现代        |
| 注重安全合规            | **Hapi**     | 安全机制强          |
| 想一体化开发（像 Laravel） | **AdonisJS** | 全家桶体验          |
| 快速生成后台            | **SailsJS**  | MVC 生成式框架      |
